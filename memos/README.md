# 政治理论种子概念数据集

## 项目概述

本项目将政治理论题库数据转换为符合 MemCube 项目要求的种子概念数据集，用于概念图迭代扩增。

## 文件说明

### 📁 数据文件

- `transformed_political_data.json` - 转换后的政治理论QA数据集 (12,092条)
- `political_seed_concepts.json` - 政治理论种子概念 (11,027个，JSON格式，含元数据)
- `political_seed_concepts.txt` - 政治理论种子概念 (11,027个，纯文本格式)

### 🔧 脚本文件

- `data_transformer.js` - 数据转换脚本 (将原始分类数据转换为QA格式)
- `concept_extractor.js` - 概念提取脚本 (从QA数据中提取种子概念)

## 使用方法

### 1. 使用种子概念进行概念图扩增

```python
# 读取种子概念
with open('political_seed_concepts.txt', 'r', encoding='utf-8') as f:
    seed_concepts = [line.strip() for line in f.readlines()]

# 或者使用JSON格式
import json
with open('political_seed_concepts.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
    seed_concepts = data['seed_concepts']

# 用于概念图迭代扩增
# 参考 overview.md 中的概念图扩增方法
```

### 2. 重新运行转换脚本

```bash
# 转换数据格式
node data_transformer.js

# 提取种子概念
node concept_extractor.js
```
